# 크몽 클론코딩 주요 기능

---

- **기능** **1: CRUD**
    - 로그인, 로그아웃 상태를 구별하여 Create, Edit, Delete 구현
    - Redux toolkit으로 전역상태관리
    - Axios로 서버와 API 통신
    
- **기능** **2: 로그인**
    - 로그인, 로그아웃 상태를 구별하여 Create, Edit, Delete, MyKmong 접근 권한 설정
    
- **기능** **3: 회원 가입**
    - 아이디 중복 확인
    - 비밀번호 중복 확인
          
- **기능** **4: 파일 올리기**
    - 단일 파일을 formData를 활용하여 서버에 전달하고, Back-end에서 S3로 upload
